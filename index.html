My Professional Work

Rage of Pluto

Rage of Pluto is a mobile physics based side-scrolling action arcade game developed for the android platform on the Google Play Store. The purpose of the game is to traverse space in search of Pluto’s long lost friend Neptune. Players will use the Moon ball and dash mechanics to traverse the enemies and obstacles.

My responsibilites coming into this project was to optimize the game as it had poor performance. I was able to invesitgate and implement design patterns such as object pooling to drastically improve the performance for all mobile devices. I also developed several gameplay features that involed the physics and interactions of the moonball weapon!

Neurobody

Neurobody is a encyclopedia application that consist of fully 3D exercises for the user to analyze and understand. These exercises are full of information to educate the user such as Instructions, tips, common avoidances, and muscle engagement.

I was the full-stack developer for this project. I was responsible for taking all the visuals (models, animations, UI) and developer the features and user flows. The main feature is the timeline that is use to traverse through the animation and stop at any frame. All data is stored and retreived using the firebase sdk for unity.

This app is currently in Beta!

Cover Me Pro is a work scheduling web application that allows managers tp schedule shifts to their staff. This also alows employees to view and manager thier shifts. They are able to trade with eachother, or request vacation/sick leave. This application also handle overtime callouts by utiling text notifcations to all eligible employees.

This application is currently in development which I am responsible for designing all user flow and features and implementing them using React and Material UI. All server side logic utlize firebase functions that handles authentication, database, and file storage.

Chatnonymous

Chatnonymous is a chat web application that allows users to communicate anonymously. This was design to create an environment where people can share their thoughts together without the concern of your identity attached to these thoughts. This allows discussions to happen at the clearest level so everyone in that environment can comprehend and understand.

I develop this after graduating from Fanshawe College. There is both a client (React) and a server (NodeJS) that uses WebSocket communication. All information is stored in the server, meaning no data is stored to keep users anonymous.
