My Professional Work

Rage of Pluto

Rage of Pluto is a mobile physics based side-scrolling arcade game developed for the android platform on the Google Play Store. The purpose of the game is to traverse space in search of Pluto’s long lost friend Neptune. Players will use the Moon ball and dash mechanics to fight through obstacles and enemies.

My responsibilites coming into this project was to optimize the game. I was able to invesitgate and implement design patterns such as object pooling to drastically improve the performance for all mobile devices. I also developed several gameplay features that involed the physics and interactions of the moonball weapon!

Neurobody

Neurobody is a encyclopedia application that consist of fully 3D rendered exercises for the user to analyze and understand. These exercises are full of information to educate the user such as Instructions, tips, common avoidances, and muscle engagement.

I was hired as the full-stack developer for this project. I was responsible for taking all visual assets (models, animations, UI) and develop all features of this application. One main feature is the timeline that is use to traverse through the animation and stop at any frame. All data is stored and retreived using the firebase sdk for unity.

This app as been ongoing development and is currently in Beta!

Rocket Recover

Rocket Recover is a mobile obstacle avoidance game on both Android and iOS. Players can choose from multiple ships with unique stats and powers to try to reach the highest scores. Customization of the environment (Background, obstacles, music) give the player a personal experience.

I was the primary game designer and developer of this game which was released under Cat Splat Studios. I work on most of the game mechanics and the deployment for the google play store and the Apple iOS store. This game also included services like leaderboards and ads.

Although it is no longer on both stores, this was a great accomplishmnet for me as it was my first publication as a primary designer and developer. You can view the gameplay and also code structure on my Github Profile.

Cover Me Pro

Cover Me Pro is a work scheduling web application that allows managers to effectively schedule their staff's shifts. This also alows employees to view and manager thier own shifts. They are able to trade with eachother, or request vacation/sick leave. This application also handle overtime callouts by utiling text notifcations to all eligible employees.

I was hired for designing all user flow/features. These are developed using React, TypeScript, and Material UI. All server side logic utlize firebase cloud functions that handles authentication, database, and file storage.

My Personal Projects

Chatnonymous

Chatnonymous is a chat web application that allows users to communicate anonymously. This was design to create an environment where people can share their thoughts together without the concern of identity attached to the conversation. This allows discussions to happen at the clearest level so everyone in that environment can comprehend and understand without judgment.

I develop this app after graduating from Fanshawe College. There is both a client (React) and a server (NodeJS) that uses WebSocket communication. All information is handled on the server, meaning no data is stored on any database to keep all user's identities and conversation private and within the moment.

Capstan Gate

Capstan Gate is a stealth horror video game that was a final student project that was made by our term. Players play as a patient that must escape from the asylum while their mind begins to loose control with disturbing visuals and persuasive voices within their head.

I was appointed Technical Lead for this project with a team of 3 programmers. My responsibilities were to lead the team and create/maintain development expectations. With those responsiblites, I also developed both the player's main mechanics and the enemy AI's perception and pathfinding sequences.
